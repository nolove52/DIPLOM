Добавлена регистрация пользователей на сайте VSKturn.

Что изменено:
1. В окне входа появилась вкладка «Создать аккаунт».
2. Пользователь вводит имя, email, пароль и повтор пароля.
3. Аккаунты сохраняются в localStorage по ключу vskturnUsers.
4. После регистрации пользователь автоматически входит в личный кабинет.
5. Вход выполняется по email и паролю.
6. Для администратора оставлен демонстрационный доступ:
   email: admin@vskturn.ru
   пароль: admin

Изменённые файлы:
- src/App.js
- src/components/LoginModal.jsx

Проект успешно собран командой npm run build.
